# Vezgo Python SDK Examples

This directory contains example implementations demonstrating how to use the Vezgo Python SDK.

## Available Examples

### [basic/](./basic/)

Basic examples including:
- **`basic.py`** - Command-line script demonstrating core SDK functionality
- **`flask_server.py`** - Flask web server with REST API endpoints

See [basic/README.md](./basic/README.md) for setup instructions.

## Getting Started

1. Install the Vezgo SDK:

```bash
pip install vezgo
```

2. Get your API credentials from [Vezgo Dashboard](https://vezgo.com/dashboard)

3. Navigate to an example directory and follow the README instructions.
